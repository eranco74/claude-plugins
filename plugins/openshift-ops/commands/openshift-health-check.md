# OpenShift Cluster Health Check

Perform a comprehensive health check of my OpenShift cluster.

Please check:
1. Cluster version and update status
2. All cluster operators (looking for degraded/progressing states)
3. Node health and resource usage
4. Pods that are not running or succeeded
5. Recent events and alerts
6. Certificate status
7. Resource availability (CPU, memory, storage)

Provide a summary of the cluster health and flag any issues that need attention. Use commands from the openshift-debugging skill.
